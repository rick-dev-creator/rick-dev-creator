## Tech Stack

```text
Backend       .NET 8 | ASP.NET Core | .NET Aspire | Blazor
Frontend      TypeScript | React | Astro
Architecture  Clean Architecture | Modular Monolith | DDD | CQRS | Event Sourcing

```

## What I Build

**Enterprise Systems** — Multi-tenant platforms with modular architecture, domain events, and CQRS patterns.

**Architecture Templates** — Reusable .NET templates for jumpstarting Modular Monolith projects with Clean Architecture.

**Real-time Dashboards** — Full-stack applications with Blazor, external API integrations, and analytics.

**Event-Driven Services** — Pub/Sub messaging with .NET Aspire and distributed systems patterns.
